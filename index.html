Pretty Things - E-commerce Website

E-commerce

Pretty Things is an e-commerce website that I built for a local clothing and accessories store that runs in Perth and St Andrews. Originally it was built in wordpress, but then switched to shopify to sell online.

Prior to the website, the business only sold goods in-store. The owner felt there was a need for an online presence to keep up with the growth in technology and how other local businesses were switching to online.

The goals that were set out for the store were as follows:

  • Make the website responsive
  • Sell goods online and track orders
  • Track stock in-store and online
  • Increase sales and traffic
  • Marketing and SEO optimized

Database Normalization toolkit

Web Application

This was the project I created for my Dissertation: Database Normalization Learning Toolkit (Finding out why users struggle with normalization). It is the first of its kind to be browser based using HTML, CSS, JavaScript and JSON.

The scope and objectives for the project were as follows:

  • Identify the specific objectives of the study + take research gathered and create a toolkit based on it.
  • This includes the key points of Database Normalization that users are struggling on.
  • Create and test different tutorials based on the research gathered. Research being user data.
  • Consistently apply updates to the toolkit until a solution was found.
